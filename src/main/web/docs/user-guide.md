# Textus Blog User Guide

Textus Blog provides public post browsing, author post management, tag
navigation, and optional user notification badges when the notification
component is assembled into the subsystem.

Use **Public blog** to browse published posts. Use **My posts** to create,
edit, publish, or update your own posts. Tags are shared in the Blog tag
space and can be entered as dot-separated paths such as `phase22.jobs`.
